AcademicReviewer 测试计划与步骤

49 项测试 · 9 个测试组 · 预计单人 2~3 小时 / 多人并行 1 小时

📋 目的：让团队同事能够按照明确步骤逐一验证系统各项功能，确保上线前所有模块稳定可用。
⏱ 预计耗时：单人完整约 2~3 小时（含等待 LLM 响应）；多人并行约 1 小时。
💰 成本提示：组 2 和组 3 消耗 API 费用，建议先用最便宜的模型跑通。校准引擎（组 7）大部分零成本。
